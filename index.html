About this dashboard

This is a Cowork live artifact. Three things are happening behind the scenes:

1. The spec sheet (memory). Hardware values — inverter count, battery kWh, observed genset fuel burn — live in a markdown file the analysis reads on every run. Keeps recommendations grounded.

2. This dashboard (live view). Persistent page. KPIs and commentary regenerate from whatever data exists. State stored locally; survives across sessions.

3. The 11:59 p.m. scheduled task (automation, pending). Once we wire up Claude in Chrome to access DeyeCloud, the task runs nightly, captures the day's totals, and pre-fills this form for your 10 a.m. review. Until then, paste values from DeyeCloud manually.

What you do each morning:

Confirm today's rooms-occupied number at the top. Glance at the four big KPIs. Read the engineer's commentary. If anything is off (production below baseline, genset cost spike, low minimum SOC), the commentary explains why and what to change.

Drag the ⠿ handle on any section to rearrange. Layout saves automatically.
